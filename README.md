# gym-super-mario-bros

## Installation

The preferred installation of `gym-super-mario-bros` is from `pip`:

```shell
pip install gym-super-mario-bros
```

## Usage

### Python

This can be run directly from the terminal without setting up any environment or installing anything else.
Just run:

```shell
python3 learning.py
```
or
```shell
python3 mario_locate_objects.py
```
